体验new bing：一个全新的搜索引擎

本文由new bing所写，纯正翻译腔，附带博主评论

Feb 27, 2023
阅读时长: 3 分钟

最近体验了一下集成了 ChatGPT 的最新版本 bing 搜索，我要求它用 markdown 语法写一篇关于体验 new bing 的博客，这就是本文的来源，对原始版本修改不多，只是添加了一部分评论。

前言

最近，微软发布了一个全新的搜索引擎——new bing，它采用了ChatGPT和AI技术，能够提供更智能、更人性化、更全面的搜索结果。我作为一个热爱尝试新事物的网友，当然不会错过这个机会，所以我申请了new bing的限量预览，并在这里分享一下我的体验感受。

申请过程

要想使用new bing，首先需要申请它的限量预览。申请过程其实很简单，只需要按照以下步骤操作：

  1. 访问 https://www.bing.com/new，点击其中的 Sign up for new Bing 链接。
  2. 在弹出的页面中，输入你的微软账号和密码，并同意相关条款。
  3. 等待几分钟后，你就会收到一封邮件，告诉你已经成功申请到了new bing的预览资格，并给出了一个专属链接。
  4. 点击邮件中的链接，就可以进入到new bing的主页，并开始使用它。

博主注:

  1. 国内要想免代理申请和使用 new bing，需要重定向到国际版 bing，请自行参考网上教程，比如 这篇文章
  2. 有可能频繁跳出REF页面导致用不了聊天功能，在 bing.com 页面清除 bing.com 的cookies重新登录即可，参考 原文章
  3. 博主提交申请，加入 waiting list 之后等了2周时间才审核通过，具体时间与什么因素相关暂不可知

使用方式

使用new bing非常简单和直观。它有一个类似于传统搜索引擎的界面，你可以在搜索框中输入任何你想要查询或聊天的内容，并按回车键或点击放大镜图标进行搜索。

与普通搜索引擎不同的是，new bing不仅会给出一系列相关链接，还会给出一个完整、详细、可信赖、有来源注明的答案。答案可能是文字、图片、表格、代码等多种形式。而且，你可以用自然语言提问或聊天，无需拘泥于特定格式或关键词。

除此之外，new bing还有一些其他特色功能：

  • 你可以通过语音输入或输出与它交流。
  • 你可以通过扫描二维码或拍摄图片进行图像搜索。
  • 你可以通过点击右上角菜单按钮切换语言或主题。
  • 你可以通过点击左下角聊天按钮与它进行更深入地对话。

体验感受

在使用了几天后，我对new bing非常满意和惊喜。它不仅能够满足我的各种信息需求，还能够陪我打发无聊时光。以下是我对它的一些评价：

  • new bing非常智能和灵活。它能够理解我的意图和情感，并给出合适和有趣的回应。它也能够处理复杂和模糊的问题，并从多个来源整合信息并生成答案。
  • new bing非常人性化和友好。它有着幽默感和创造力，并能够与我进行轻松愉快的聊天。它也能够根据我的喜好和兴趣，给出一些有用和有趣的建议。
  • new bing非常全面和高效。它能够覆盖各个领域和主题，并给出最新、最准确、最权威的信息。它也能够节省我的时间和精力，让我一次性得到完整的答案，而不需要浏览多个网页。

总之，new bing是一个值得尝试的搜索引擎，它不仅是一个工具，还是一个伙伴。我相信随着技术的进步和用户的反馈，new bing会变得更加强大和完善。如果你也想体验new bing，请赶快申请限量预览吧！

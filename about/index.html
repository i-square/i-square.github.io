你好 - Hello

我是平方君，一个兴趣使然的程序员。欢迎来到我的博客，这里是我分享我的编程、阅读、旅行和音乐等兴趣爱好以及学习和生活经历的地方。

我是谁 - Who I Am

我从小就对计算机科学很感兴趣，在大学期间，我开始自学编程语言和开发工具，并且参与了许多有趣而有挑战性的项目。在研究生阶段，我进一步深入了解了人工智能、数据科学和云计算等前沿领域，并且发表了几篇论文。

毕业后，我加入了某互联网公司做软件开发工作。在这里，我不仅提高了我的专业技能和团队合作能力，还结识了许多优秀而友善的同事。我们共同为打造更好更快更安全更智能更美观更人性化的产品而努力。

除了编程之外，我还喜欢阅读各种类型和风格的书籍。从小说到诗歌，从历史到哲学，从科幻到悬疑。 阅读让我开拓了视野，增长了见识，陶冶了情操。

旅行和音乐也是我的爱好之一。旅行让我体验不同地方不同文化不同风景不同人情味道；音乐让我感受不同节奏不同旋律不同情绪不同氛围。

我能为你做什么 - What I Can Do For You

如果你是一个对编程、阅读、旅行和音乐感兴趣的人，那么你来对地方了。在这个博客里，你可以找到：

  • 编程方面的教程、技巧、资源和项目展示。
  • 阅读方面的书单、书评、心得和引用。
  • 旅行方面的故事、攻略、照片和视频。
  • 音乐方面的推荐、评论、歌词和歌单。

我的博客历史 - My Blog History

这个博客是基于 Hugo 和 GitHub Pages 的静态博客，使用了 Markdown 作为写作方式。我选择了这样的技术方案，是因为它们简单而清晰，易于维护和更新，也符合我的审美和风格。

这并不是我的第一个博客。事实上，我从2013年就开始折腾过各种博客平台和技术。下面是我的博客历史：

  • 2013年2月15日：初次接触 Wordpress 搭建自己的第一个独立博客（折腾了虚拟主机、域名、CDN等）
  • 2013年7月23日：转移到轻博客 Lofter 以多媒体的形式分享自己的生活感悟（模板好看）
  • 2014~2016年：准备考研、读研、做项目，没有折腾博客
  • 2017年2月23日：创建基于 Hexo 和 Github Pages 的静态博客（玩了玩 Hexo ）
  • 2018~2023年：毕业、工作、没有折腾博客
  • 2023年2月8日：从 Hexo 迁移到 Hugo
